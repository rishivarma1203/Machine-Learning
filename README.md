# Explainable Machine Learning with Decision Trees

This repository contains a tutorial demonstrating how machine learning models can be interpreted using feature importance and SHAP explanations.

## Overview

The tutorial trains a Decision Tree classifier on the Titanic dataset to predict passenger survival.  
Explainable AI techniques are then used to understand how the model makes predictions.

The analysis includes:

- Decision tree visualization
- Feature importance analysis
- SHAP summary plot
- SHAP waterfall explanation for individual predictions

These techniques help improve transparency and interpretability of machine learning models.

## Requirements

Python libraries required:

numpy  
pandas  
matplotlib  
seaborn  
scikit-learn  
shap  

Install using:

pip install numpy pandas matplotlib seaborn scikit-learn shap
